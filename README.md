# MERN Auth App

A full-stack authentication application built with the MERN stack featuring secure user registration, login, JWT authentication, and protected routes.

## Live Demo

🔗 https://mern-register-and-login-frontend.vercel.app/login

## Features

* User Registration & Login
* JWT Authentication
* Protected Routes
* Password Hashing with bcrypt
* Form Validation & Error Handling
* Responsive UI

## Tech Stack

**Frontend:** React, React Router, Axios, Tailwind CSS
**Backend:** Node.js, Express.js, MongoDB, Mongoose, JWT, bcrypt
